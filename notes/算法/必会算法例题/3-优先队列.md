底层实现方式：堆或二叉搜索树。

## 1. 数据流中第 k 大的元素

[Leetcode - 703 Kth Largest Element in a Stream (Easy)](https://leetcode.com/problems/kth-largest-element-in-a-stream/)

```java
final PriorityQueue<Integer> q;
final int k;
public KthLargest(int k, int[] nums) {
    this.k = k;
    q = new PriorityQueue<>(k);
    for(int n : nums){
        add(n);
    }
}

public int add(int val) {
    if(q.size() < k){
        q.offer(val);
    }else if(q.peek() < val) {
        q.poll();
        q.offer(val);
    }
    return q.peek();
}
```

## 2. 滑动窗口的最大值

[Leetcode - 239 Sliding Window Maximum (Hard)](https://leetcode.com/problems/sliding-window-maximum/)

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    if(nums == null || k <= 0) return new int[0];
    int[] res = new int[nums.length - k + 1];
    int ri = 0;
    
    Deque<Integer> q = new ArrayDeque<>();
    for(int i = 0; i < nums.length; i++){
        // 删除前面超出滑动窗口的元素
        while(!q.isEmpty() && q.peek() < i - k + 1){
            q.poll();
        }
        // 删除比 i 小的元素
        while(!q.isEmpty() && nums[q.peekLast()] < nums[i]){
            q.pollLast();
        }
        q.offer(i);
        if(i >= k - 1){
            res[ri++] = nums[q.peek()];
        }
    }
    return res;
}
```
