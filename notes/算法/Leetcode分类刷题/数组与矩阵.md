
# 将数组中的 0 移动到后面

[Leetcode - 283 Move Zeros (Easy)](https://leetcode.com/problems/move-zeroes/)

```java
public void moveZeroes(int[] nums) {
    int index = 0;
    for (int num : nums) {
        if (num != 0) {
            nums[index++] = num;
        }
    }
    while (index < nums.length) {
        nums[index++] = 0;
    }
}
```
