# IntelliJ IDEA 工具

## 常用快捷键

Ctrl + Z	撤销

Ctrl + Shift + Z	重做（撤销冲正）



Ctrl + F	当前文件检索

Ctrl + Shift + Z	全文检索



Ctrl + R	当前文件查找替换

Ctrl + Shift + R	全文检索替换



Double Shift	全局查找

Alt + F7	查找当前这个方法被调用的地方

Ctrl + Alt + B	快速定位当前方法的实现



Alt + Insert	插入构造、get/set、overriding、实现接口



Ctrl + D	复制当前行并且粘贴到当前行之下

Ctrl + Y	删除当前行



注释

Ctrl + /

Ctrl + Shift + /



Ctrl + Alt + L	格式化

Ctrl + Alt + O	优化 Import



Alt + Enter	自动导入包

Shift + F6	重命名文件
